Markdown 学习笔记

概述

Markdown 旨在实现[易读易写]的网络书写语言. Markdown的灵感来源是纯文本电子邮件.

Markdown并不是HTML的取代品或替换品, Markdown的语法仅对应了HTML标记的一小部分,不属于Markdown涵盖范围的标签, 都可以直接在文档中使用HTML书写, 需要注意的标签例如<div>, <table>, <pre>, <p>必须在前后加上空行与其他内容区隔开, 且他们的开始标签和关闭标签不能使用制表符和空格来缩进.

区块元素

段落和换行

一个 Markdown 段落是由一个或多个连续的文本行组成, 它的前后要有一个以上的空行, 普通段落不该用空格或制表符来缩进.

上述特性指出Markdown允许段落内的强迫换行(插入换行符), 如果确实需要依赖Markdown来插入<br>标签, 需要在插入出输入两个空格然后回车.

标题

Markdown支持两种标题语法:类Setex和类atx形式.

类Setex形式使用底线,利用=(最高节标题)和-(第二阶标题). 任何数量的=和-都有效.

类atx形式则是在行首插入1到6各#, 对应标题1到6阶.

区块引用

Markdown 标记区块引用(Blockquotes)是使用类似email中 > 的引用形式. 在每行最前加上>

This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.

或者在段落第一行前加上>也是可行的.

Ab quae minim tamen quamquam, quem litteris sempiternum.Aut arbitror
coniunctione. Senserit illum consequat laborum. Doctrina ubi quis.

区块引用可以嵌套, 也可以使用其他的Markdown代码.

列表

Markdown支持有序列表和无序列表.
无序列表使用*, +, -作为标记:

  • Red
  • Green
  • blue

有序列表则使用数字标号和英文句号:

  1. Beijing
  2. Shanghai
  3. Guangzhou

代码区块

使用制表符或者4个空格就可以建立代码区块, Markdown会使用< pre>或者<code>标签将代码区块包装.

分割线

用三个以上的 * , - , _ 来建立分割线, 分割线内不能有其他内容.

区段元素

链接

Markdown支持两种形式的链接语法: 行内式和参考式. 链接文字使用方括号来标记.

行内式链接: 方括号后面紧接圆括号并插入网址链接即可, 如果还要加上链接的title, 只要在网址后使用双引号把title文字包起来即可, 例如:
This is an inline-style link.

参考式链接: 链接文字的括号后面接上另一个方括号, 而在第二个方括号里面要填入用以辨识连接的标记:
This is a reference-style link .
然后在任意处将这个标记(id)的链接内容定义出来
[id]: http://www.google.com 'google'

隐式链接标记功能让你可以省略指定链接标记，这种情形下，链接标记会视为等同于链接文字，要用隐式链接标记只要在链接文字后面加上一个空的方括号，如果你要让 “Google” 链接到 google.com，你可以简化成：
[Google][]
然后定义链接内容:
[Google]: http://www.google.com

强调

Markdown 使用星号( * )和底线( _ )作为强调标记, 被其包围的内容会转成<em>标签包围, 用两个的话, 会被转成<strong>. e.g.:
this is a emphasize example
this is a strong example

代码

如果要标记一小段行内代码, 可以使用反引号( ` )将他包起来. e.g.: prinf()

图片

图片标记同样分为行内式和参考式.
行内式语法: ![Alt text](/path/to/img.jpg)
参考式语法: ![Alt text][id]
定义方式: [id]: url/to/img.jpg "title"

其它

反斜杠

Markdown中可以利用反斜杠来插入其它有意义的符号(如星号,方括号等). e.g.: *, [

自动链接

Markdown支持简短的链接处理, 用方括号包起来则会自动转成链接. e.g.:
[http://google.com]
