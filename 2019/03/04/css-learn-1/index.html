ref: http://biaoyansu.com

概述

CSS(cascading style sheet), 定义样式和兼顾交互, 将样式与html标签抽离, 避免了标记元素和结构元素的混乱.
结构化有以下优点:

  • 利于内容索引
  • 可访问性好
  • 实现高级页面表现
  • 易于维护
